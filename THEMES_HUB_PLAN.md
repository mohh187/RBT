# استوديو التصميم الموحد + نقلة نظام الثيمات والقوالب (خطة + 100 اقتراح + فحص الجودة)

> الحالة (2026-07-05): Liquid Glass افتراضي للمنصة بمنزلقات شدة (شفافية/لزوجة/تموج) تسري على كل شيء عبر متغيرات CSS؛ «يتبع العام» أصلح حجب المزج للهوية؛ قاعدة التباين: كل ثيم يعيد تعريف الرموز — ممنوع لون سطح/نص ثابت في أي ثيم.
> H1 منفذ (الاستوديو موحد). H2 منفذ: معرض القوالب المرئي (#36) + ثيم أورورا (#17) + زجاجي داكن دائم (#16) + محرر الثيم المخصص (#21/#22) + زجاج المنيو فوق أي سكن + زجاج مخصص لكل نطاق (glassFxBy). المتبقي: H3 (الذكاء 71-80) و H4 (الجودة 86-99) وبقية الثيمات (18-20) والموسمية (24-25).

## 1) المشكلة التنظيمية والحل: «استوديو التصميم» مكان واحد
التشتت الحالي: هوية عامة + ثيم نظام + KDS في «إعدادات المنشأة»، وسكنات المنيو + الكاشير + القفل في «تصميم المظهر»، والفاتورة تبويب ثالث، والشاشات قسم رابع.
**الحل**: تبويب واحد `الاستوديو` يستبدل «تصميم المظهر» ويضم كل شيء بهيكل ثابت: يمين = **معاينة حية واحدة** تتبدل تلقائياً (منيو/كاشير/مطبخ/قفل/فاتورة/شاشة عرض/بروفايل) بأجهزة (كمبيوتر/تابلت/جوال)؛ يسار = شجرة أقسام:
1. **الهوية** (الشعار/الألوان/الخط/الأرقام) — ينبوع واحد تكسر منه كل الثيمات.
2. **ثيم النظام** (الثيمات + قوة الزجاج + مزج الأقسام).
3. **المنيو** (السكنات والعناصر والخلفيات — البطاقات الحالية تنتقل كما هي).
4. **الكاشير والمطبخ** (قوالب + ثيم + خلفية).
5. **شاشة القفل**. 6. **الفاتورة**. 7. **شاشات العرض**. 8. **البروفايل والاستوري** (أنماطها).
تنفيذ تدريجي بلا كسر: التبويب الجديد يعيد استخدام البطاقات الحالية (نقل JSX لا إعادة كتابة)، وروابط «انتقلت إلى الاستوديو» في الأماكن القديمة لدورة واحدة.

## 3) قائمة الفحص الدقيق (تُنفذ بعد كل تغيير ثيمي)
لكل ثيم (كلاسيكي/زجاجي/زجاجي-داكن/نوار/دافئ/حاد) × وضع (فاتح/داكن): □ أزرار الثلاثة الملونة نص مقروء □ الرقاقات والتبويبات □ الحقول وplaceholder □ الشارات الست □ الجداول/الصفوف □ التوست والشيتات (المنبثقات عبر body) □ POS كامل (بطاقات/فاتورة/أسعار) □ KDS (تذاكر/KPI/أعمدة) □ شاشة القفل بكل أطوارها + خلفية صورة □ الويزارد □ المعاينات الثلاث تعمل تفاعلياً □ لا لون ثابت جديد في CSS (grep `rgba(255` داخل كتل الثيمات) □ الأداء: لا blur فوق 44px ولا تراكب زجاج أكثر من 3 طبقات.
