# لقطات النظام الحقيقية — Landing gallery

تظهر هذه اللقطات تلقائياً في قسم «لقطات حقيقية من النظام» بالصفحة الرئيسية.
قبل وجودها يبقى القسم مخفياً تماماً (لا صور وهمية إطلاقاً).

## الطريقة المُوصى بها: التقاط آلي آمن

كلمة المرور تبقى في ملف محلي مستثنى من Git (`.env.shots`) — لا تُرسل في المحادثة
ولا تُرفع لأي مكان، والسكربت يقرأها عند التشغيل فقط.

```
cp .env.shots.example .env.shots      # ثم املأ الإيميل/الباسوورد/السلَق فيه
node scripts/shots.mjs                # أضف SHOTS_HEADED=1 لمشاهدة المتصفح
```

يسجّل الدخول لموقعك المباشر ويلتقط: `dashboard.jpg` · `cashier.jpg` ·
`themes.jpg` · `menu.jpg` — وتظهر فوراً في الصفحة. بعدها احذف `.env.shots`
(أو غيّر كلمة المرور) للأمان.

## أو ضع الملفات يدوياً

| الملف | ماذا تصوّر | مقاس مقترح |
|------|-----------|-----------|
| `dashboard.jpg` | لوحة التحكم / الإدارة | ~1600×1000، أفقي |
| `menu.jpg`      | المنيو على الجوال (`/m/<slug>`) | ~800×1600، عمودي |
| `cashier.jpg`   | شاشة الكاشير POS | ~1600×1000، أفقي |
| `themes.jpg`    | شاشة الثيمات/التخصيص | ~1600×1000، أفقي |

أسماء الملفات والتسميات قابلة للتعديل من مصفوفة `shots` في `RealGallery`
بملف `src/routes/Landing.jsx`.
