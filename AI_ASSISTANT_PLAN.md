# خطة المساعد الذكي (المدير الافتراضي) — Gemini — MenuLink

> الهدف: حقل محادثة يعطيه المدير أوامر بلغة طبيعية، فيفحص حالة النظام (المخزون/الطلبات/العملاء/الإعدادات) **ويُنفّذ** ويعدّل أي شيء (إعدادات، أصناف، مخزون، وصفات، عضوية…) بصلاحية كاملة وضوابط أمان. يُربط بـGemini API لاحقاً؛ الآن نجهّز **البنية التحتية**.

## 1) البنية (3 طبقات)

1. **سجل الإجراءات `lib/actions.js`** — كتالوج موحّد من الدوال القابلة للاستدعاء، كلٌّ بـ JSON Schema (name, description, params) يلفّ دوال `db.js` الموجودة:
   - قراءة: `getInventoryStatus`, `listLowStock`, `listOrders`, `getCustomer`, `getSalesSummary`, `getSettings`…
   - كتابة: `addMaterial`, `receiveStock`, `setRecipe`, `updateItem`, `setItemAvailability`, `updateSettings`, `grantMembership`, `createOffer`, `voidOrderItem`…
   كل إجراء: `{ name, description, params(JSONSchema), risk:'safe'|'confirm'|'danger', run(args, ctx) }`.

2. **لقطة السياق `buildContext()`** — تُعطي النموذج أرضية: عدد المنتجات، نواقص المخزون، مبيعات اليوم، الطلبات النشطة، الإعدادات الحالية، عدد العملاء/الأعضاء. تُمرّر مع كل رسالة.

3. **جسر Gemini `lib/aiBridge.js`** — يرسل الرسالة + schemas الإجراءات كـ Gemini **function tools** → يستقبل function calls → ينفّذها عبر السجل → يعيد النتائج للنموذج → يكرّر حتى الإنجاز (agent loop). مفتاح `VITE_GEMINI_API_KEY`.

## 2) واجهة المحادثة
- لوحة محادثة (للمدير فقط) برسائل + إدخال + بثّ.
- تعرض **الإجراءات المنفّذة** كبطاقات قابلة للمراجعة، وتطلب **تأكيداً** للعمليات `confirm`/`danger` (حذف، تعديل أسعار جماعي، تصفير مخزون).

## 3) الأمان والصلاحيات
- محصور بدور المدير/المالك.
- **سجل تدقيق** لكل إجراء AI (مَن/ماذا/متى/قبل-بعد).
- **معاينة (dry-run)** للعمليات الجماعية قبل التنفيذ.
- حدود معدّل + إيقاف طارئ (kill switch).

## 4) التجهيز الآن (قبل ربط Gemini)
1. توحيد دوال `db.js` لتكون نقيّة وقابلة للاستدعاء برمجياً (مُنجز غالباً).
2. بناء `lib/actions.js` (السجل + schemas + risk).
3. بناء `buildContext()`.
4. هيكلة سجل التدقيق `tenants/{tid}/aiLog/{id}`.
5. ترك نقطة ربط واضحة لـ`aiBridge` (Gemini) — يُفعّل عند توفّر المفتاح.

> النتيجة: عند ربط Gemini لاحقاً، لا نعيد بناء المنطق — فقط نوصل السجل + السياق بـfunction-calling. كل دوال النظام جاهزة كأدوات.
