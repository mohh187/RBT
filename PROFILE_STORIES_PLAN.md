## 1) بروفايل المنشأة — «صفحة تعريفية حيّة»

### 1.1 الفكرة والقيمة
صفحة عامة `/m/{slug}/about` تصل من أيقونة في هيدر المنيو (تُخفى عبر آلية HIDEABLE الموجودة — نضيف مفتاح `profile`). تعمل كمدونة/سجل حي: أخبار، فعاليات نُظّمت، شخصيات زارت المكان، إنجازات، فيديوهات — **أرشيف يبني الثقة ويرفع بقاء الزبون داخل المنيو**.

### 1.2 نموذج البيانات
```
tenants/{tid}/posts/{id}: {
  type: 'news' | 'event' | 'visit' | 'announcement' | 'video' | 'gallery',
  titleAr/En, bodyAr/En (نص غني مبسّط: فقرات + روابط),
  media: [{ kind: 'image'|'video', url, thumb?, w?, h? }],
  pinned, published, publishAt?, createdAt, updatedAt,
  likes: 0, views: 0,                    // عدّادات مجمعة
  tags: ['رمضان','ضيوف'],
}
tenants/{tid}/posts/{id}/comments/{cid}: { name, text, deviceId, at, hidden }
```
- **التخزين**: `uploadImage/uploadFile` الموجودة (Storage) — فيديو بحد حجم بحسب الخطة (مثلاً 50MB مجاني / 200MB برو)، بدون تحويل ترميز (المتصفح يشغّل mp4/webm مباشرة).
- **القواعد**: قراءة عامة للمنشور `published`; كتابة المنشورات لطاقم المنشأة (صلاحية إدارة الإعدادات أو صلاحية جديدة `MANAGE_CONTENT`); تعليقات: إنشاء عام بقيود (طول/معدل) وحذف/إخفاء للطاقم.
- **الإدارة**: قسم «المحتوى» في اللوحة: قائمة منشورات + محرر (عنوان/نص/وسائط متعددة/نوع/تثبيت/جدولة نشر) + إحصاءات مشاهدات وإعجابات.
- **العرض العام**: تبويبات (الكل/أخبار/فعاليات/زيارات/فيديو) + منشور مثبّت أعلى + مشاركة رابط المنشور.

## 2) الاستوريات (نمط إنستغرام) على المنيو

### 2.1 التجربة
دوائر استوري أعلى المنيو (تحت الهيدر) — تُفعّل/تُخفى من الإعدادات (`hidden: stories` + مفتاح تشغيل). فتح ملء الشاشة: تقدّم تلقائي بشريط زمني علوي، لمس يمين/يسار للتنقل (RTL-native)، ضغط مطوّل يوقف، سحب لأسفل يغلق — نفس قواعد الحركة المعتمدة (transform/opacity فقط).

### 2.2 نموذج البيانات والدورة
```
tenants/{tid}/stories/{id}: {
  kind: 'image' | 'video', url, thumb?, caption?, link? (صنف/عرض/منشور),
  createdAt, expiresAt (افتراضي +24h، قابل للتمديد أو "مثبّت" دائم في Highlights),
  views: 0, likes: 0, active,
  highlight?: 'رمضان'   // مجموعات مثبتة كألبومات دائمة
}
tenants/{tid}/stories/{id}/reactions/{deviceId}: { like, emoji?, at }
tenants/{tid}/stories/{id}/replies/{rid}: { text, name?, deviceId, at, hidden }
```
- **الانتهاء**: الاستعلام العام يجلب `active && expiresAt > now` (فلترة عميل + عامل تنظيف يومي في Functions — scheduled function نمط auditRetention الموجود).
- **الرفع والتصوير المباشر**: `<input type="file" accept="image/*,video/*" capture="environment">` يفتح الكاميرا مباشرة على الجوال — بلا مكتبات. فيديو ≤30 ثانية (فحص مدة عبر عنصر video قبل الرفع) و≤25MB.
- **التفاعل**: إعجاب/إيموجي/رد نصي بلا تسجيل عبر deviceId (نفس نمط `customer.js` للتقييمات) — الردود تصل للوحة كـ«صندوق ردود الاستوري» مع إخفاء/حظر.
- **القواعد**: قراءة عامة للنشط؛ إنشاء/حذف للطاقم؛ reactions/replies إنشاء عام بمعدل محدود، إدارة للطاقم.
- **القياس**: views تُزاد مرة لكل جهاز (localStorage)؛ لوحة إحصاءات لكل استوري (مشاهدات/إكمال/ردود).
- **الربط الذكي**: زر «أضف للاستوري» من: صنف (بطاقة جاهزة بصورته وسعره)، عرض/كود خصم، منشور بروفايل، وحتى «طبق اليوم من الكاشير». والاستوري بدوره يحمل زر سحب لأعلى → الصنف/العرض المربوط.
- **الحدود بحسب الخطة**: مجاني = 3 استوري نشطة صور فقط؛ برو = غير محدود + فيديو + Highlights.

### 2.3 مراحل التنفيذ (الاثنان معاً)
- **P1 (الأساس)**: posts CRUD + صفحة /about + دوائر الاستوري + عارض ملء الشاشة (صور) + إعجاب + إظهار/إخفاء + قواعد.
- **P2**: فيديو + تعليقات/ردود مع الإشراف + Highlights + التصوير المباشر + إحصاءات.
- **P3**: الجدولة والقوالب والمساعد الذكي والتكاملات (شاشات العرض/الولاء/النشرة) + المئة فكرة تباعاً حسب الأولوية التجارية.

### 2.4 مخاطر ومعالجات
| الخطر | المعالجة |
|---|---|
| محتوى مسيء في التعليقات/الردود | فلتر كلمات + إخفاء افتراضي حتى الموافقة (خيار) + حظر deviceId + حدود معدل في القواعد |
| تضخم التخزين بالفيديو | حدود حجم/مدة بحسب الخطة + تنظيف المنتهي + ضغط الصور الموجود |
| بطء المنيو | الاستوري lazy: تُجلب الدوائر فقط (thumbs)، والوسائط عند الفتح؛ posts لا تُحمّل في مسار المنيو إطلاقاً |
| خصوصية الضيوف في صور الفعاليات | تنبيه موافقة في المحرر + أداة طمس وجه بسيطة لاحقاً |
| كسر الواجهات الحالية | كل شيء مسارات ومجموعات جديدة؛ المنيو يتغير بسطر شرطي واحد (شريط الدوائر) خلف مفتاح |
